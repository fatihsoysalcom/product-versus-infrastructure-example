# Product versus Infrastructure Example

This example demonstrates the conceptual difference between a 'Product' and its underlying 'Infrastructure' in a digital ecosystem. The `UserManagementProduct` provides direct value to end-users (registration, login), while relying on `DatabaseService`, `LoggingService`, and `NotificationService` as its essential, non-user-facing infrastructure components.

## Language

`python`

## How to Run

Save the code as `main.py`.
Run from your terminal: `python main.py`
